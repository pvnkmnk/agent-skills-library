# Music Project Planner

Plans music production and release projects — EP, album, single, DJ mix release — with phased milestones, task breakdowns, and Obsidian integration.

## Quick Start

Invoke via your agent: `Invoke music-project-planner` (OpenCode/Codex) or `/skill music-project-planner` (Freebuff).

## What it does

- Takes a project type (EP, album, remix, live set) and target release date.
- Breaks the project into phases: Concept → Production → Mixing → Mastering → Release → Promotion.
- Generates a task list per phase with estimated effort.
- Maps tasks to a calendar with buffer time built in.
- Saves the plan to `Projects/Music/project-title.md` in Obsidian with checkboxes and milestone markers.

## Dependencies

- Obsidian vault with write access
- Optional: Obsidian Tasks plugin for live task tracking
